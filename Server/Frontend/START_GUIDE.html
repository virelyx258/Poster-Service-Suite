Poster系统启动指南

这个指南将帮助您正确运行Poster登录注册系统。系统包含以下文件：

  • index.php - 前端页面，包含登录和注册表单
  • api_handler.php - 后端API处理文件
  • README.md - 项目说明文档
  • START_GUIDE.html - 本启动指南
注意：在使用之前，请确保您的计算机上已安装PHP环境。

步骤1：安装PHP

1访问 PHP官方网站 下载适合您操作系统的PHP版本。
2按照安装向导完成PHP的安装。
3安装完成后，将PHP安装目录添加到系统环境变量中。

步骤2：验证PHP安装

1打开命令提示符（Windows）或终端（macOS/Linux）。
2输入以下命令并按回车： 
php -v
如果显示PHP版本信息，说明安装成功。

步骤3：启动PHP内置服务器

1在命令提示符或终端中，导航到Poster系统所在的目录： 
cd c:\Users\hi\Desktop\Poster_Web
2启动PHP内置服务器： 
php -S localhost:8000
成功启动后，您将看到类似以下的输出： 
PHP 版本信息 Development Server (http://localhost:8000) started

步骤4：访问系统

1打开浏览器。
2在地址栏输入以下URL并按回车： 
http://localhost:8000
您将看到Poster登录注册系统的界面。

测试模式说明

由于邮件发送功能可能受到网络或SMTP服务器配置的限制，当前系统使用了测试模式：

  • 在注册页面，您可以输入任意有效的邮箱地址
  • 对于验证码，直接输入 123456 即可通过验证

常见问题排查

问题1：启动服务器时提示"php不是内部或外部命令"

解决方法：检查PHP是否正确安装，以及是否已添加到系统环境变量中。

问题2：登录时返回undefined或网络错误

解决方法：

  1. 打开浏览器的开发者工具（按F12）
  2. 切换到"控制台"（Console）选项卡
  3. 再次尝试登录，查看控制台中的错误信息
  4. 检查api_log.txt文件，查看详细的请求和响应日志

问题3：无法连接到远程API服务器

解决方法：检查您的网络连接，确保可以访问 https://www.rsv.ee 网站。

日志文件说明

系统会生成以下日志文件，帮助排查问题：

  • api_log.txt - 记录所有API请求和响应信息
  • raw_input.txt - 记录原始的请求输入数据
  • email_log.txt - 记录邮件发送相关的日志（如果启用）
