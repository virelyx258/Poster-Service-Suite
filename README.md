## 简介
> [!IMPORTANT]
> 以下内容于2025年11月8日撰写，本软件的实际功能可能会因新版本推送而与简介有所差异。

**Poster 远控服务套件** 是与 Xiaomi Vela JS 快应用 **Poster** 配套的`远程控制服务套件`。用户可将其自行部署到云服务器，随后即可通过手表端向云服务器发送 POST 请求而实现简单远程控制的效果。

- 技术栈：`易语言`
- 支持的操作系统：Windows NT 4.0 or Later

## 编译

该软件使用 易语言 5.95 进行编写和调试。如果你想要编译该软件，请准备以下环境：

- Windows 7 和以上版本的电脑
- 易语言 5.95 及更新版本

同时，该软件引用了 [精易模块](https://ec.125.la)，请在编辑 / 引用代码时遵守相关协议。
